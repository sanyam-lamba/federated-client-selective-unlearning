# Federated Client Unlearning in Medical Imaging

Implementation of Federated Client Unlearning (FCU) framework with Advanced 
Sharding strategies for privacy-preserving machine unlearning in healthcare.

## Tech Stack
PyTorch, DenseNet121, MedMNIST, Python

## Key Features
- Model-Contrastive Unlearning (MCU)
- Frequency-Guided Memory Preservation (FGMP)
- Shard-level selective forgetting
- Benchmarked FFT, DFT, DWT, FWT transforms
